Please make sure to follow the following procedure:

  1. Select the used syringe volume.
  2. Select the wanted dose to be disbursed.
  3. Select the number of times you would like the dose to be disbursed in how many hours
  4. The device will calculate the remaining syringe volume and the doses administered.
  5. Please make sure to double-check the dose before clicking SUBMIT. Stay Safe, Stay Healthy!

  • Syringe Volume [5ml, 10ml, 20ml]:

                                                                  ml
